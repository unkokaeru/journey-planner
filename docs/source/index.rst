Installation
------------

To install journey-planner, simply run:

.. code-block:: bash

    pip install journey_planner

Usage
-----

After installation, you can use the package by importing it in your project:

.. code-block:: python

    import journey_planner

Or, you can run the package as a script:

.. code-block:: bash

    python -m journey_planner
